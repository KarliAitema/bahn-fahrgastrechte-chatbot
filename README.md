# Bahn Fahrgastrechte Chatbot

Ein einfacher Chatbot, der Bahnfahrern Auskunft über ihre Fahrgastrechte bei Verspätungen und Zugausfällen gibt.

## Funktionen
- Informationen zu Entschädigungssätzen (25% ab 60 Min, 50% ab 120 Min).
- Anleitung zur Beantragung (Online, App, Formular).
- Liste der benötigten Unterlagen.

## Installation
```bash
python3 chatbot.py
```

## Tech Stack
- Python 3
- Regex-basiertes Keyword-Matching
